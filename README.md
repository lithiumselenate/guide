# UTCSSA 指南

本网站基于 [VuePress v2 (beta)][vuepress-v2] ，使用 Cloudflare Pages 自动构建并部署。

- UTCSSA 新生手册
- UTCSSA 选课指南

## 贡献

请 Fork 此代码仓库并提交 Pull Request ，包管理器请使用 [Yarn 1 (Classic)][yarn] 。

```sh
# 安装依赖
yarn

# 开启开发服务器
yarn dev

# 使用 Prettier 格式化代码
yarn format
```
